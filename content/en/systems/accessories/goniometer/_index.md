The goniometer is an attachment for Bridge12 EPR probes, such as the [Bridge12 XLP](https://www.bridge12.com/products/x-band-pulsed-epr-endor-resonator/). The accessory is attached to the top of the probe and allows the user to rotate the sample stick around its axis. The goniometer accessory can be fully remote controlled from a terminal window. A Python package is available to allow for complete integration into the spectrometer software.
