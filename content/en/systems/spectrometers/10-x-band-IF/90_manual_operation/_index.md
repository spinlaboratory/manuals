To manually operate the X-Band IF system please install the following software:

* **Microwave Synthesizer:** Please download and install the [Software Control GUI](https://windfreaktech.com/wp-content/uploads/RF-Signal-Generators/software/SynthHD3_2a.zip) distributed by [Windfreak Technologies](https://windfreaktech.com). 

* **Digital/Analog Controls:** Please download [Kippling](https://cdn.docsie.io/file/workspace_u4AEu22YJT50zKF8J/doc_VDWGWsJAhd453cYSI/boo_9BFzMKFachlhscG9Z/file_0VQecfNudEDcCn8dq/labjack-2019-05-20.exe) distributed by [LabJack](https://labjack.com).

Once these two software packages are installed, the X-Band IF system can be completely remote controlled.
