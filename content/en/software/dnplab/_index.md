[SpinLab](https://spinlaboratory.com/) is an object-oriented [Open Source](https://en.wikipedia.org/wiki/Open_source) Python-based package for importing, processing, and analyzing data recorded in a Magnetic Resonance experiment (EPR, NMR, DNP). The aim of the project is to provide a free, turn-key python-based processing package DNP-NMR data.

SpinLab is distributed free of charge under the [MIT License](https://en.wikipedia.org/wiki/MIT_License).

For more information:
* Visit the SpinLab [Online Documentation](https://spinlaboratory.com/)
* Check out the [DNPLab GitHub Repository](https://github.com/spinlaboratory/spinlab)
* Please report all issues on the [DNPLab GitHub Issue Tracker](https://github.com/spinlaboratory/spinlab/issues)
