Welcome to the online documentation of [Open VnmrJ](https://openvnmrj.org/) for the Bridge12 Single Channel NMR (SCN) spectrometer. Open VnmrJ is an open-source software package designed for the acquisition, analysis, and processing of NMR data.

Bridge12 uses [Open VnmrJ](https://openvnmrj.org/) to control the Bridge12 SCN spectrometer. For Overhauser-DNP enhanced NMR spectroscopy, the [Bridge12 Microwave Power Source (MPS)](https://www.bridge12.com/products/microwave-power-source-for-odnp-spectroscopy/) can also be controlled through OVJ.

The documentation provided here covers basic experiments, mainly focusing on the acquisition of Overhauser-DNP enhanced <sup>1</sup>H NMR spectra.

For an in-depth documentation of OVJ please visit the [Open VnmrJ](https://openvnmrj.org/) website or consult the pdf documentation installed with the software.
